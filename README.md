N7geeks workshop 12/02/2020 - Casino Dapp
# Run : #
 Casino Dapp using trufflesuite, Reactjs, to run the app follow these steps:
 install ganache or any eth testnet provider and run it
 then
 run:
 * git clone https://github.com/didolehbal/N7geeks_Casino_Dapp myfolder
 * cd myfolder
 * npm i truffle -g
 * truffle migrate
 * cpy build/contracts/* webapp/contracts
 * cd webapp && npm i
